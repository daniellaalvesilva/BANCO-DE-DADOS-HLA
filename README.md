# Padronização de bancos de dados imunogenéticos

Este script visa a limpeza, organização e padronização de banco de dados relacionado a informações imunogenéticas (HLA), com o uso da linguagem de programação Python (versão 3.12.12).

Foi implementado um prompt com a descrição de cada etapa do script. Ambos foram desenvolvidos com apoio de ferramentas de Inteligência Artificial (ChatGPT – OpenAI, modelo GPT-5), com revisão e validação manual dos autores. Esses códigos inicialmente, foram desenvolvidos para o banco de dados de 2020 (bd_2020), e aplicado para os demais anos.
